# Personal_web
New website built on [HUGO](https://gohugo.io/getting-started/quick-start/) with [hugo-profile](https://github.com/gurusabarish/hugo-profile) template.

## cheatsheet for hugo
```bash
# install hugo (mac)
brew install hugo

# start server
hugo server -D

# stop server
Ctrl+C
```

## Upload to the github
```bash
git add *file* && git commit -m "*note*" && git push
```
